## 技术栈

+ Go
+ SocketIO
+ MongoDB
+ Redis


## Start

```bash

git clone --recurse-submodules https://github.com/peanut996-im/im-backend.git

# or
git clone https://github.com/peanut996-im/im-backend.git
cd im-backend
git submodule update --init

```

## Update And deploy

```bash
bash update-deploy.sh
```
