# peanut996.im

## 概述

这里是Peanut996.IM的开源代码库, 对当前的稳定版本的代码库进行预览查看。

## 技术栈

### 数据库

+ MongoDB
+ Redis

### 后端

+ [go-socketio](https://github.com/googollee/go-socket.io)
+ [Gin](https://github.com/gin-gonic/gin)

### 前端

+ Vue全家桶
+ TypeScript
